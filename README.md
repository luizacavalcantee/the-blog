<h1 align=center> THE BLOG. </h1>

## Como usar
Você pode clonar esse repositório e rodar: 
    ```
    npm install
    ```
<br>
<br>
E depois: 
    ```
        npm run dev
    ```
<br>
<br>
Ou você pode ver a implantação do projeto agora mesmo: [https://the-blog-dev.vercel.app/]


## Sobre 
Blog feito em 48 horas (02/09-04/10).
